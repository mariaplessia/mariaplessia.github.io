Edge Detection

Implementing a custom Sobel Filter and using OpenCV's Canny Filter in Python





Custom Sobel Filter

For the custom Sobel Filter, I preprocessed the images by changing them to grayscale. Then, I implemented the Horizontal Sobel filter, which consists of a horizontal derivative filter and a vertical Gaussian filter.

Then, I implemented the Vertical Sobel filter, which consists of a vertical derivative filter and a horizontal Gaussian filter.

Finally, I combined the vertical and horizontal Sobel images to produce the final Sobel-filtered output image and inverted it so that the background is white and the edges are black.

Canny Filter

I preprocessed the images by denoising with the application of a 5x5 Gaussian blur, and then changing them to grayscale. Then, I initialized the threshold and aperture values, defined the slide bars for them, and initialized the image.

Using an indefinite loop that breaks using esc, I updated the threshold and aperture values for every iteration of the loop and dynamically renewed the image in the GUI to reflect the new changes, while also inverting it so that the background is white and the edges are black.
